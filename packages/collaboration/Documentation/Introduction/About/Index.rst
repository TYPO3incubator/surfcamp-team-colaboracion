What does it do?
================

The Collaboration extension (`EXT:collaboration`) aims to improve the editor experience in the TYPO3 Backend by providing tools that facilitate teamwork and prevent conflicts.

**Key Features:**

- **Contextual History Button**: Quickly see when a page or record was last edited and by who, directly from the button bar in the Page or Workspace modules.
- **Real-time Notifications (SSE)**: Stay informed about important events like cache clearing or when another user starts editing the same record you are working on.
- **Enhanced Internal Notes**: The `sys_note` records are extended to allow assigning notes to specific backend users, making it easier to manage tasks and communication within the TYPO3 Backend.
- **Record Locking Awareness**: Better visibility and handling of locked records to avoid overwriting changes.
