.. _internalNotes:

==============
Internal Notes
==============

The Collaboration extension enhances the standard TYPO3 internal notes (`sys_note`).

Assigning Users
===============

When creating or editing an internal note, you can now assign it to a specific backend user.

1.  Open or create an **Internal Note**.
2.  Use the **Assign user** dropdown to select a colleague.
3.  Save the record.

Assigned notes can be easily identified in the List module or when viewing notes on a page, facilitating task management within the backend.
